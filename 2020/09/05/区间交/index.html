区间交

题目链接

Problem

选取一些区间，使其区间交不小于\(k\)。 # Solution 区间交即是选出的区间的\([L-max,R-min]\)。将所有区间按左端点排序，枚举每一个区间(区间的\(r-l >= k\))将其左端点作为\(L-max\)。设之前右端点不小于\(L-max+k\)的区间有\(a\)个，对答案的贡献是\(2^a\)。可以用数据结构(比如支持单点修改和区间查询的线段树)或者优先队列维护。 # Code
